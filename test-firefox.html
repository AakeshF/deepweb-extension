DeepWeb Firefox Extension Test Page

Test Instructions

  1. Make sure the DeepWeb extension is installed in Firefox
  2. Click the extension icon or press Ctrl+Shift+Y to toggle the chat
  3. Select some text and right-click to use "Ask DeepWeb AI"
  4. Test different AI models (Chat, Coder, Reasoner)

Sample Text for Testing

Artificial Intelligence is transforming the way we interact with technology. From natural language processing to computer vision, AI technologies are becoming increasingly sophisticated.

Here's a simple Python function to test with DeepSeek Coder: 

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
